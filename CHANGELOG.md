## v1.01 (2026-05-12)

### 新增
- 生词录入：输入俄语单词，自动调用 DeepSeek 获取中文翻译和例句
- 句式录入：输入俄语句子，自动修正语法并给出中文翻译和关联句式
- 单词库：搜索、浏览已录入的所有单词，支持删除
- 句式库：搜索、浏览已录入的所有句式，支持删除
- 复习模式：三选一选择题，支持俄→中、中→俄、句式翻译三种题型
- 加权随机出题：优先考察错误率高的条目，全对的低频但不遗忘
- DeepSeek API 集成：翻译、例句生成、语法纠错（使用 Flash 模型节省成本）
- API Key 持久化：设置后自动保存，重启无需重新输入
- PyInstaller 打包：构建独立 .app，双击即用，无需安装任何环境
- Schema 版本迁移机制：后续升级数据库结构不丢已有数据
- 数据目录分离：数据库存 `~/.russian-wordbook/`，替换 .app 不影响数据
- API Key 设置后自动补译已有未翻译单词
